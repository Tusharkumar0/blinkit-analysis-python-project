# 🛒 Blinkit Sales Analysis using Python

## 📖 Overview

This project performs **Exploratory Data Analysis (EDA)** on the Blinkit grocery sales dataset using Python.
The analysis focuses on understanding sales performance, product categories, outlet characteristics, and customer purchasing trends through data cleaning, statistical analysis, and visualizations.


## 🎯 Objectives

* Clean and preprocess the dataset
* Perform Exploratory Data Analysis (EDA)
* Analyze sales across product categories and outlet types
* Identify key sales trends and business insights
* Create meaningful visualizations for decision-making


## 🛠️ Tech Stack

* **Pandas**

## 🔍 Key Business Insights

* 💰 Total sales generated were **₹12,01,681.48**.
* 🥬 **Fruits & Vegetables** emerged as the highest-selling product category.
* 🍟 **Snack Foods** ranked as the second-best performing category.
* 🏪 **Medium-sized outlets** generated the highest sales.
* 🌍 **Tier 3 locations** contributed the maximum revenue among outlet locations.
* 🥗 **Low Fat products** accounted for approximately **64.6%** of total sales.
* 📊 Correlation analysis was performed to understand relationships between numerical features.
